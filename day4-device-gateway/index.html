Day4 StudyFlow S3 智能终端

把 Day3 已完成的本地 Agent Core 接到真实 CoreS3 / ESP-Claw / 微信入口，完成“掌上设备触发 Day3 Agent Core”的云边协同 Demo。

Step 1

启动设备网关

运行 run_day4_service.command，把 Day3 Agent Core 包装成 OpenAI-compatible 服务。

Step 2

配置 S3

ESP-Claw 填入 http://192.168.1.196:8000/v1 与模型 pocket-campus-agent。

Step 3

语音演示

说“请整理今天资料”，再说“确认整理”，最后说“给我一道复习题”。

核心入口

查看 README Day3/Day4 边界 设备配置单 本机 Dashboard

服务配置

OpenAI Base URL: http://192.168.1.196:8000/v1
Model: pocket-campus-agent
API Key: classroom-demo-key
Health: http://127.0.0.1:8000/health
Dashboard: http://127.0.0.1:8000/dashboard

交付证据

运行后检查 day4_agent_workspace/ 与 day4_trace_log.jsonl。这些文件记录设备输入、二次确认、Day3 Agent Core 写文件、复习卡生成和最终返回。
