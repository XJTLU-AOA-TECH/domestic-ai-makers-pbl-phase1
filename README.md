# XJTLU AOA 国产AI动手派

> **XJTLU AOA Domestic AI Hands-On**
> AI-PBL Course · Phase 1 · Domestic AI Technology Foundations

---

## 这是什么？

四天，从零开始走通大模型工程化的完整链路。不是听课，不是看排行榜——你自己动手。

| 天 | 主题 | 你要做什么 |
|---|---|---|
| **Day1** | 模型评测与路由 | 不看排行榜，你做裁判。选模型、出题、打分、出路由表 |
| **Day2** | 能力模块构建 | 把选好的模型封装成 Prompt / Schema / Eval |
| **Day3** | 桌面 Agent | 写一个能读文件、检索知识、调用模型的 ReAct Agent |
| **Day4** | 设备接入 | 刷 ESP-Claw（龙虾），按实体按钮触发你的 Agent |

---

## 怎么用？

1. `git clone` 整个仓库
2. 每天进入对应文件夹，先读 `README.md`
3. 按里面的步骤操作——可以不跑参考代码，但交付文件名和格式必须一致
4. 每一天的产物交给下一天，形成完整链路

---

## 你需要什么？

- Python 3.8+
- 一个国产大模型的 API Key（任意平台）
- M5Stack CoreS3（Day4 需要）
